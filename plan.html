Notice d'Utilisation

1. Importer une liste d'élèves

Pour remplir automatiquement les tables, le programme utilise le contenu de votre Presse-papiers (Copier/Coller).

Méthode 1 : Depuis PRONOTE
1. Allez dans l'onglet Ressources > Classes (ou Emploi du temps).
2. Affichez la liste des élèves de la classe.
3. Sélectionnez la liste (Ctrl+A ou à la souris) et faites Copier (Ctrl+C).
Méthode 2 : Depuis Excel / Tableur
1. Créez une colonne avec vos élèves.
2. La première ligne doit impérativement contenir le mot "Nom".
3. Les lignes suivantes contiennent les noms des élèves.
4. Sélectionnez la colonne et faites Copier.

Ensuite, cliquez sur le bouton "✨ Répartir les élèves" dans l'application.

2. Manipuler les tables et élèves

  • Ajouter une table : Glissez-déposez depuis le menu de gauche vers la zone blanche.
  • Déplacer une table : Cliquez sur la bordure noire de la table et glissez.
  • Sélectionner plusieurs tables : Tracez un rectangle de sélection sur le fond blanc avec la souris.
  • Échanger deux élèves : Glissez simplement une case élève vers une autre case.
  • Modifier un nom : Cliquez sur une case pour éditer le texte manuellement.

3. Sauvegarde et Export

  • 💾 Enregistrer le plan : Télécharge un fichier de>.json contenant votre disposition.
    Le fichier est enregistré dans votre dossier "Téléchargements" par défaut.
  • 📄 Exporter en PDF : Génère un PDF A4 paysage optimisé pour l'impression.
